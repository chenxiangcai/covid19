# 重庆理工大学课程设计（Covid-19）

## 1. 需要初始化2次依赖包：
   - covid19根文件夹下 `npm i`
   - server文件夹下  `npm i` 
   
## 2. server文件夹下启动app.js (启动后台服务)

## 3. 在covid19根文件夹下 `npm start`
